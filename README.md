# YouTube Auto RU Subtitles

Chrome-расширение, которое в один клик включает субтитры на YouTube и автоматически переводит их на русский язык.

## Как работает

Расширение использует content script для взаимодействия с DOM плеера YouTube:

1. Включает субтитры (если они отключены).
2. Открывает меню настроек плеера.
3. Переходит в раздел «Субтитры».
4. Выбирает пункт «Перевести».
5. Устанавливает язык перевода — «Русский».

Так как YouTube — это SPA, логика реализована асинхронно с использованием `async/await` и задержек для корректного ожидания перерисовки меню.

## Технологии

- Chrome Extension (Manifest V3)
- Content Scripts
- DOM automation
- Async / Promises API

## Назначение

Избавляет от необходимости вручную проходить через несколько уровней меню для включения авто-перевода субтитров.
